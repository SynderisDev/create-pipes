# Changelog

## 0.1.2

- Changed the item pipe recipe to `copper sheet, dried kelp, copper sheet`.

## 0.1.1

- Rebuilt public jar after repository cleanup.
- Renamed the internal Java package to `com.synderisdev.createpipes`.
- Updated public mod metadata author to `SynderisDev`.

## 0.1.0

- First public alpha setup.
- Adds item pipes with Normal, Pull, Push, and None side modes.
- Adds Create filter support for pull and push sides.
- Adds a Network Control Tool for turning connected pipe networks on or off.
- Offline networks stop ticking.
- Steam is not required by default. Steam-powered balancing is planned.
